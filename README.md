# Image Watermarking
Image Watermarking using Discrete Cosine Transformation on MATLAB.

## Getting Started

How to use
```    
git clone https://github.com/aakashjhawar/Image-Watermarking.git
cd Image-Watermarking
```
(Execute the main.m file) 
## Prerequisites

- MATLAB
- Simulink
